## How to install and run

`git clone https://github.com/bbullek/ReactJS-Minitab-Sample.git`
`cd react_minitab`
`npm install`
`npm start`

This runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
